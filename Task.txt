Необходимо разработать:
1. Форма данных для отображения следующей информации:
a. Имя;
b. Фамилия;
c. Должность;
d. Год рождения;
e. Зарплата.
2. Форму «Сотрудники»:
a. Внизу расположить таблицу, на которой будет отображена вся информация (Имя, Фамилия, Должность, Год рождения, зарплата);
b. Вверху расположить фильтр, который позволит фильтровать информацию по должности;
c. Справа расположить группу кнопок:
i. Добавить сотрудника (с помощью которой можно будет добавлять информацию в таблицу);
ii. Удалить сотрудника;
iii. Отчет (по нажатию на которую, необходимо будет отобразить отчет);
3. Разработать отчет, желательно с помощью SSRS, но можно и просто в Excel, который будет показывать среднюю зарплату для каждой должности.
4. Технические требования:
a. Winforms;
b. Язык разработки: С#, VB.NET;
c. База данных SQL Server (2008 и выше); 
d. Доступ к базе осуществлять через ADO.NET.
